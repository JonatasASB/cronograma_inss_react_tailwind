# Cronograma INSS PRD — React + Tailwind

## Rodar localmente
```bash
npm install
npm run dev
```

## Build de produção
```bash
npm run build
npm run preview
```

## Correções de UI/UX aplicadas
- contraste forte nos selects
- texto do dropdown visível
- foco visível nos campos
- labels mais claras
- áreas de clique maiores
- layout melhor em mobile e desktop

## Hospedagem grátis
### Vercel
- conectar repositório GitHub
- framework: Vite
- build command: `npm run build`
- output: `dist`

### Netlify
- conectar repositório GitHub
- build command: `npm run build`
- publish directory: `dist`

### Cloudflare Pages
- conectar repositório GitHub
- framework preset: Vite
- build command: `npm run build`
- output directory: `dist`

Os dados são salvos no navegador usando localStorage.
